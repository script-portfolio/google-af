
# google-af

Google auto fill

Scirpt to get google search machine answer prompt lines when
we fill google search text field

!"Screenshot on google page](/google-autofill.png)

Run:
    ### apple - default value

    vlad@dnestr:~/src/google-af$ python google-af.py apple

        apple store

        apple id
